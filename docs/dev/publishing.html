This guide outlines the steps to build and upload the package to PyPI.

1. Install build tools

python -m pip install --upgrade build twine

2. Build distributions

python -m build

3. Upload

Upload the generated files in dist/ to PyPI:

twine upload dist/*

Consider uploading to --repository testpypi first to verify the package.
