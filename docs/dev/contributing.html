How to Contribute

  1. Fork the repository on GitHub.
  2. Create a new feature branch: git checkout -b feature/my-feature
  3. Make your changes and commit with descriptive messages.
  4. Push your branch: git push origin feature/my-feature
  5. Open a Pull Request describing your changes.

Coding Style

We follow standard Python conventions (PEP8). Please format your code accordingly.

Running Tests

Run tests with:

pytest tests/

Commit Message Format

Use clear and concise commit messages. Example:

Fix: corrected context vector normalization bug

Code of Conduct

Please respect all contributors and follow the project's code of conduct.
