The Contextual AI system is composed of the following main layers:

1. Input Layer

Receives raw messages or event logs for processing.

2. Context Providers

  • RedisContextProvider: Stores and retrieves context vectors from Redis.
  • FileBasedContextProvider: Local file storage for development and testing.
  • FileContextProvider: Persists context entries to a JSON file for offline use.
  • SimpleContextProvider: Minimal in-memory provider for examples.

3. Core Processing Modules

  • Fuser: Combines similar messages.
  • FuzzyDeduplicator: Eliminates duplicates.
  • Categorizer: Categorizes messages.
  • CacheManager: Manages caching with invalidation.
  • AIInference: Executes AI decision engines.

4. Distance Engine

Calculates semantic distances between context vectors using cosine or Euclidean metrics.

5. Learning Layer

Manages machine learning models for better categorization and decision-making.

6. Output and Integration

Exposes APIs and integrates with other systems or frontends.
