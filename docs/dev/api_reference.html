Core Interfaces and Classes

ContextProvider (interface)

Abstract interface to fetch context data from different sources.

class ContextProvider:
    def get_context(self, key: str) -> dict:
        pass

    def set_context(self, key: str, data: dict) -> None:
        pass

Core Modules

  • Fuser: Combines similar messages based on context distance and merge logic.
  • FuzzyDeduplicator: Removes duplicates based on hash or semantics.
  • Categorizer: Categorizes messages using rule-based or AI models.

CacheManager

Provides caching with invalidation support.

Example Usage

from providers.redis_context_provider import RedisContextProvider
from core.fuser import Fuser

provider = RedisContextProvider()
fuser = Fuser()

context_data = provider.get_context('example_key')
fused = fuser.fuse([context_data])
print(fused)
